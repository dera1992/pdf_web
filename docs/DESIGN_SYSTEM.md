# Design System

## Design Tokens
### Color Palette
- **Primary Red**: #E53935
- **Dark Red**: #B71C1C
- **Light Red**: #FFCDD2
- **Neutral 900**: #1B1B1B
- **Neutral 700**: #4F4F4F
- **Neutral 300**: #E0E0E0
- **Neutral 100**: #F5F5F5

### Typography
- **Primary font**: Inter
- **Heading sizes**: 32 / 24 / 20 / 16
- **Body sizes**: 14 / 12

### Spacing
- 4, 8, 12, 16, 24, 32, 48

### Radii
- 4, 8, 12

## Components
- **Buttons**: primary (red), secondary (outline), tertiary (text).
- **Inputs**: focus ring with red accent.
- **Sidebar panels**: 280px width default, collapsible.
- **Cards**: soft shadows, 8px radius.

## Dark Mode
- Invert document colors while keeping UI elements high-contrast.
- Red accents remain for active controls.

## Figma Style Guide
- Create a shared library with tokens and components.
- Include page templates: landing, viewer, collaboration.
